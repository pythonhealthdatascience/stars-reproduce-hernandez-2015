13.26-13.35, 13.44-14.03, 15.17-15.21: Adding parallel processing

I tried switching the loop in Experiment1.py into a parallel loop, to speed up the run time.

First, I just changed it to a function with a loop.

I ran this with the same parameters as above (10 pop 1 gen 1 run) to confirm the results didn‚Äôt change at all due to the parallel processing and, indeed, they remained the same, so this is successfully implemented. The only change was - as expected - to the times files, which is now a single file.
